Esse projeto consiste no trabalho 2 de IPI

O projeto contém 2 arquivos python, um para cada questão.

# Questão 1
O arquivo questao1.py contém a resolução da questão 1 e para executá-lo basta executar o comando:
```bash
$ python3 questao1.py 
```
Ele irá identificar o tumor na imagem brain.jpg.

# Questão 2
O arquivo questao2.py contém a resolução da questão 2 e para executá-lo basta executar o comando:
```bash
$ python3 questao2.py 
```
Ele irá segmentar a imagem onion.jpg para diferentes quantidades de clusters K-means.
